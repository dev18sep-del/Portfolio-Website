About Me

A little bit about my background, interests, and goals

I'm Dev Patel, a Computer Science student at a private university in India, currently in my second year of a BTech program (2023–2027). From the moment I wrote my first line of code, I knew software development was where I belonged.

My journey began with C++ and data structures, then expanded into web development — building interfaces with HTML/CSS/JS, creating REST APIs with Node.js, and working with databases like MongoDB and MySQL.

My goal is to become a skilled Full Stack Software Engineer — writing clean, efficient code and contributing to products that matter.
